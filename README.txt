Safety Generator - README
This is the README file for the Safety Generator, a web application designed to provide users with random safety tips and encourage safety awareness.

Features:

Generates random safety tips displayed on the page.
Includes a "GENERATE SAFETY" button to refresh the displayed safety tip.
Features a link to the Occupational Safety and Health Administration (OSHA) website for further safety resources.
How to Use:

Open the index.html file in a web browser.
Click the "GENERATE SAFETY" button to display a random safety tip.
Click the OSHA logo image to visit the OSHA website for more safety information (opens in a new tab).
Technology Stack:

HTML: Provides the structure of the web page.
CSS: Styles the visual elements of the page.
JavaScript: Manages user interaction and retrieval of safety tips (implemented in the script.js file).
Customization:

The current safety tips are hardcoded in the JavaScript file. You can modify the script to:

Add your own custom safety tips.
Retrieve safety tips from an external data source (e.g., an API).
